
# GESTION NHR

Application de gestion de stocks multisite pour l'industrie hôtelière (Bassam et Abidjan).

## Fonctionnalités

- **Tableau de bord** : Vue d'ensemble des stocks et alertes.
- **Stocks (Inventory)** : Liste des produits avec niveaux de stock par site.
- **Entrées (Achats)** : Saisie des réceptions de marchandises.
- **Sorties (Besoins)** : Demandes de produits par les services (Cuisine, Bar, etc.).
- **Multi-site** : Gestion centralisée pour Bassam et Abidjan.

## Technologie

- React + TypeScript (Vite)
- Tailwind CSS (Styling)
- Supabase (Backend & Auth)
- Lucide React (Icons)

## Installation

1. Cloner le projet.
2. Installer les dépendances :
   ```bash
   npm install
   ```
3. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```

## Structure

- `/src/components` : Composants réutilisables.
- `/src/pages` : Pages principales.
- `/src/lib` : Configuration Supabase et utilitaires.
- `/src/types` : Types TypeScript.
